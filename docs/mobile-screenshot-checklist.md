# Mobile Screenshot Checklist (Phase 15J)

Capture each screen at three widths before launch:

- **390 px** — iPhone 14 / 15 portrait
- **414 px** — large Android portrait
- **768 px** — tablet portrait

For each width, capture both light and dark color schemes if both are
supported.
